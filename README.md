# 舞萌DX账户信息查询工具

一个用于查询舞萌DX游戏账户信息的Android应用。

## TODO
- [ ] 查询B50


## 功能特性

- 📱 获取舞萌DX账户UserID
- 🔍 查询用户基本信息
- 📊 查看详细游戏数据
- 🎮 支持二维码码快速登录

## 使用方法

1. 在舞萌 |中二 公众号中获取二维码
2. 在App中输入二维码信息获取UserID
3. 使用UserID查询账户信息
